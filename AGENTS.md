# Imgoo — Agent guide

## Commands

```sh
npm run dev      # Vite dev server at http://localhost:5173
npm run build    # production build → dist/
npm run preview  # serve the built dist locally
```

No lint, test, typecheck, or formatter scripts exist. No TypeScript — all JS.

## Architecture

- **Single-page app**: Vue 3 (`<script setup>`) + Vite 5 + Element Plus 2 + Pinia + vue-i18n.
- **Entry**: `src/main.js` mounts `App.vue` to `#app`.
- **Single route** (`/`) in `src/router/index.js` → `HomeView.vue`.
- **Client-only**: all image compression runs in-browser via `browser-image-compression` using Web Workers. No server.

## Auto-imports (critical)

Two unplugin plugins generate global type declarations — **do not manually edit** these files:
- `src/auto-imports.d.ts` — Vue (`ref`, `computed`, `watch`, `defineComponent`, etc.), Pinia (`defineStore`, `storeToRefs`, `acceptHMRUpdate`), vue-router (`useRouter`, `useRoute`) are globally available with zero imports.
- `src/components.d.ts` — Element Plus components and local `src/components/` SFCs are auto-registered. Use any El-* component in templates without importing.

## SCSS

- Variables (`$primary`, `$text-primary`, `$bg-white`, etc.) and responsive mixins (`@include mobile`, `@include desktop`) are auto-injected globally via `vite.config.js` → `css.preprocessorOptions.scss.additionalData`.
- **Do not** `@use` or `@import` variables or responsive mixins in component styles — they are already available.
- Two breakpoints: **Mobile** ≤768px, **Desktop** >769px (see `src/assets/styles/responsive.scss`).

## Path alias

`@` → `src/`. Use `@/components/`, `@/stores/`, `@/composables/`, etc.

## i18n

- Locale persisted in `localStorage` key `'locale'`. Default: `'zh'`, fallback `'en'`.
- Translation files: `src/i18n/locales/zh.js`, `src/i18n/locales/en.js`.
- Switch with `locale.value = 'en'` in composition API.

## Batch processing constraints

| Limit | Value |
|---|---|
| Max files per batch | 20 |
| Single file size | 20 MB |
| Total file size | 100 MB |
| Concurrent processing | 5 (batch size) |

Defined in `src/stores/imageStore.js`.

## CI / Deploy

Push to `main` → GitHub Actions builds (`npm install && npm run build`) and deploys `dist/` to GitHub Pages.

## Generated / ignored

- `src/auto-imports.d.ts` and `src/components.d.ts` — auto-generated by unplugin plugins.
- `dist/`, `node_modules/`, `.env*` — in `.gitignore`.
